# 轻松拼豆

轻松拼豆是一个纯前端的拼豆图纸生成工具。用户上传图片后，应用会在浏览器本地完成图片取样、背景剔除、品牌色盘匹配、网格图纸生成、材料清单统计，并支持导出 PNG 或 PDF。

## 功能特性

- 上传 PNG、JPG、WebP 等本地图片生成拼豆图纸
- 默认图纸宽度 48 格，支持 16-96 格调整，并保持原图比例
- 内置 Perler、Hama、Artkal 三个品牌基础色盘
- 使用 Lab 颜色空间做最近色匹配
- 自动剔除透明区域，以及连接到图片边缘的白底/浅灰背景
- 生成带网格线、色号/符号标记的实体图纸
- 自动统计材料清单：品牌、色号、颜色名、数量、总豆数、建议底板数量
- 支持下载 PNG 图纸和 PDF 图纸
- 作品广场内置约 20 个示例，使用真实照片素材生成对应图纸
- 无后端、无账号系统，图片只在浏览器本地处理

## 本地运行

请先安装 Node.js 18 或更高版本。

```bash
npm install
npm run dev
```

启动后打开终端显示的地址，通常是：

```text
http://127.0.0.1:5173/
```

## 构建生产版本

```bash
npm run build
```

构建产物会输出到 `dist` 目录。

## 使用说明

1. 打开首页，点击「上传照片」。
2. 选择或拖入一张本地图片。
3. 选择 Perler、Hama 或 Artkal 色盘。
4. 调整图纸宽度，查看生成后的图纸和材料清单。
5. 点击 PNG 或 PDF 下载图纸。
6. 点击「浏览作品广场」查看真实照片示例和对应图纸。

## 背景剔除说明

工具会把以下区域作为空白处理，不计入材料清单：

- PNG 图片中的透明区域
- 从图片边缘连通进入画面的白色或浅灰背景

这适合处理白底商品图、白底宠物照片、白底植物照片等场景。例如上传一张白底汽水罐图片时，图纸只会保留汽水罐主体，而不会把整张白底都算成白色豆子。

## 色盘说明

项目内置的是适合演示和基础制作的品牌基础色盘，色号与颜色名参考品牌公开色卡信息，屏幕色值为近似值：

- Perler Bead Color Reference
- Hama Colour Chart
- Artkal Color Chart

实体豆子的真实观感会受批次、光照、屏幕校色和拍摄环境影响，最终购买和制作请以品牌官方色卡或实物为准。

## 示例素材来源

作品广场使用了 Wikimedia Commons 的真实照片素材，并保存在 `public/gallery` 中用于本地静态展示。素材包括小猫、小狗、树和花等白底或透明底对象图片。

## 项目边界

当前版本是本地静态应用，不包含后端服务、登录注册、云端上传、真实用户作品存储或在线协作功能。
